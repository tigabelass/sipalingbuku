Tentang Website

Website ini dibuat untuk membantu pengguna menemukan dan merekomendasikan buku terbaik.

Kenapa Website Ini Dibuat?

Website ini bertujuan untuk mempermudah pencarian buku berdasarkan kategori, memberikan rekomendasi, serta menampilkan informasi detail tentang setiap buku.

Fitur Utama:

  • 🔍 Pencarian Buku: Temukan buku berdasarkan judul atau genre.
  • 📖 Detail Buku: Lihat informasi lengkap, termasuk penulis dan deskripsi.
  • ⭐ Rekomendasi Buku: Dapatkan rekomendasi berdasarkan genre populer.
  • 📚 Antarmuka Sederhana: Navigasi yang mudah digunakan.

Teknologi yang Digunakan:

  • ⚡ Django – Backend & database handling
  • 💻 Bootstrap – Untuk tampilan responsif
  • 📡 Google Books API – Untuk mengambil data buku
  • 🔗 JavaScript – Untuk interaksi pengguna
